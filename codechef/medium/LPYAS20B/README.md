## Problem

### Print Pattern

Write a program to output the square of * (stars) of size 4 (check the output below).

```
  
  
  
   

```

## Solution

**Language:** Python  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-20T17:19:05.124Z  

```py
# cook your dish here
for i in range(1,5):
    for j in range(1,5):
        print("*",end="")
    print()

```
