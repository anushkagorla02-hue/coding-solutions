### Print Right Angled Triangle

Input

```
 
```

## Solution

**Language:** Python  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-20T17:04:41.573Z  

```py
# cook your dish here
for i in range(0,6):
    for j in range(i):
        print("*",end="")
    print()
```
