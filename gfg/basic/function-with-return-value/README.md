# Function With Return

![Difficulty](https://img.shields.io/badge/Difficulty-Basic-red)

## Problem

Given an integer  **n**, You need to use the return keyword to return twice the value of n.

 **Examples:** 

```
Input: n = 2
Output: 4
Explanation: 2 * 2 = 4
```

```
Input: n = 4
Output: 8
Explanation: 2 * 4 = 8
```

 **Constraints:** 
1 ≤ n ≤ 10
